AWS 身分驗證系統

PHP × MySQL × AWS EC2 + RDS | 安全登入與會話管理

專案概述

建置於 AWS 雲端環境 的安全登入系統，使用 PHP 處理表單驗證與會話管理，資料儲存於 MySQL RDS，部署於 EC2。

核心功能

  • 使用者註冊與登入
  • 密碼雜湊加密（password_hash）
  • Session 會話管理與自動登出
  • 表單驗證與防 SQL Injection

雲端架構

前端表單 → EC2 (PHP) → RDS (MySQL)，實現高可用性與資料持久化。

學習成長

掌握 後端安全開發、資料庫設計、雲端部署，具備企業級身份驗證系統實作能力。
