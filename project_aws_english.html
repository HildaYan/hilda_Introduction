AWS 英文學習平台

HTML × CSS × JavaScript × AWS EC2 + RDS + S3

專案概述

基於 AWS 雲端架構 開發的互動式英文學習平台，整合前端 UI 與後端資料庫，實現測驗、進度追蹤與響應式體驗。

核心功能

  • 互動式測驗：單字、文法、聽力即時回饋
  • 學習儀表板：圖表化進度與正確率分析
  • 響應式設計：支援手機、平板、電腦
  • 安全登入：PHP 會話管理 + 資料庫驗證

雲端架02構

使用 S3 靜態託管 + CloudFront CDN，搭配 EC2 後端 API 與 RDS MySQL 資料庫，實現高效能與全球存取。

學習成長

掌握 全端開發、雲端部署、資料庫設計，成功將概念轉化為可運行的雲端應用，具備企業級開發能力。
